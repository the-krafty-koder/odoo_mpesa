MPESA Paybill/Till Module for POS

Make payments at the POS using either a till number or a paybill number.

Starter Usage Summary

Create a new journal at the Point Of Sale

Choose Bank or Cash type and a new page will appear with the heading Point of Sale.Click on it and check the MPESA payment method and Paybill or Till from the select field that appears.

Make sure to add the new journal to the main Point of Sale so that it can be used.
